# クラウドサービス運用コスト

## IaaS

* 最低料金は高め
* スケールアウトにコストがかかる

## PaaS

* 最低料金は低め
* IaaSより開発保守コストが高い
* スケールアウトが容易

### 制限

* PaaSはsshができない
* ブロックストレージできない
* IaaSの自由度が高い
